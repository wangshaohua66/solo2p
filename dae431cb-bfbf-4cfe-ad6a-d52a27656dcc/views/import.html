导入说明
  • 支持 BrickLink 网站「Parts」导出的 CSV 格式
  • 支持 BrickLink XML Wanted List / Inventory 格式
  • 字段会自动映射：ITEMID→零件号、COLOR→颜色、QTY→数量等
  • 重复零件号会批量更新而非报错
  • 所有数据仅保存在本地 IndexedDB，不会上传服务器
  • 如需自定义字段映射，请先预览后再确认导入
